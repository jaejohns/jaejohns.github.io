Visual Reference of Liger's Components

Getting Familar with the Components

Below are screenshots of how the components will look with their "default" settings.

Navigation

Responsive navigation. Up to three dropdown levels.

Billboard

Just like the name suggests, this calls attention to an important message with a call-to-action.

Slider

Have an array of images that you can click through by using the left and right arrows.

Animated Forms

Liven up your form inputs by using these animated inputs.

Pod

Section of informative content paired with a visual on top and supporting title and text below the image.

Media Object

Similar to the pod, this has image on the left and text on the right.

Buttons

Various types of buttons with a default animated hover effect.
