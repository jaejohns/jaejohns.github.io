The Liger CSS Grid Framework

About the Framework

The Liger CSS Grid Framework is a lightweight grid system but unlike other lightweight css grid systems Liger offers great component options to achieve a feature rich website experience. It is derived from years of working in a corporate design environment. Liger also has an unique implementation method, which will allow you to only use the css styles you want or need. If you've worked with other css grids before, you know you you can inherit css styles that will make your website bloated with unused styles. With Liger, you can finally have a grid system without having all the bloated css!
